# CommHub - Vue3 + Spring Boot + MariaDB 커뮤니티 애플리케이션

완전히 작동하는 풀스택 커뮤니티 웹 애플리케이션입니다.

## 🎯 주요 기능

- 📝 **게시판**: 글 작성, 댓글, 좋아요, 검색, 카테고리
- 💬 **실시간 채팅**: 채팅방, 메시지, 멤버 관리
- 👥 **사용자 관리**: 회원가입, 로그인, 프로필

## 🛠️ 기술 스택

- **Frontend**: Vue3, Pinia, Axios, Vite
- **Backend**: Spring Boot, JPA, MariaDB
- **DevOps**: Docker, Docker Compose

## 🚀 빠른 시작

```bash
docker-compose up -d
```

Frontend:  http://localhost:5173
Backend:   http://localhost:8080/api
